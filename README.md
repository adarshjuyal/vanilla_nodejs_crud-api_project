# vanilla_nodejs_crud-api_project
This is vanilla Node.js CRUD API movie project.

# Routes
- **GET** `/api/movies`
- **POST** `/api/movies`
- **GET** `/api/movies/:id`
- **PUT** `/api/movies/:id`
- **DELETE** `/api/movies/:id`

 ***
 # Installation
 ```
 Install dependencies
 npm install

 Run in develpment
npm run dev

 Run in production
npm start
